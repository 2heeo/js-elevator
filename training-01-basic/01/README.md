# 과제 1

섭씨온도를 화씨온도로 또는 그 반대로 변경하는 자바스크립트 함수를 개발하세요.

공식:
  - Fahrenheit -> Celsius : (°F − 32) ÷ 1.8 = °C 
  - Celsius -> Fahrenheit : (°C × 1.8) + 32 = °F

예상 출력:
X°C는 Y°F와 같습니다
Y°F는 X°C와 같습니다

## 예제

```javascript
console.log(convertCelsiusToFahrenheit(1));
// 1°C는 33.8°F와 같습니다

console.log(convertFahrenheitToCelsius(1));
// 1°F는 -17.22°C와 같습니다
```
