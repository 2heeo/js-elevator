# 과제 10

날짜 객체를 받고 지정된 날짜 구성으로 반환할 수 있는 자바스크립트 함수를 개발하세요.
구하고 싶은 날짜 구성은 변수로 보내야 된다.

가능한 형식:
형식 1 = yyyy-mm-dd
형식 2 = yyyy/mm/dd
형식 3 = yyyy.mm.dd


참고:
[Date 객체](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Date)

## Example

```javascript
console.log(formatDate(date, 2));
// 2018/11/30
```
